# BeoNexus AI Architecture

Multi-Agent System for Livestock Management - Interactive Architecture Visualization

## Overview

BeoNexus is an AI-powered multi-agent system designed to provide evidence-based recommendations for livestock management, with a focus on sustainable farming practices and antibiotic reduction.

## Features

- **Multi-Agent Architecture**: Orchestrator + Specialist agents (Mastitis, Nutrition, Reproduction, Economics)
- **Transparent Reasoning**: Shows which agents contributed and their analysis
- **Data Integration**: Connects to multiple sources (Veterinary DB, Farm Records, Scientific Literature, IoT Sensors)
- **Evidence-Based**: All recommendations backed by scientific data and expert validation

## Tech Stack

- React 19
- Lucide React Icons
- Webpack 5
- Node.js
